Navigate to the Project Directory: Change into the project's root directory in your terminal or command prompt.

1. cd WeatherApp

Run the Application

2. dotnet run

3. Access the Application: 

Open a web browser 
Enter Localhost URL: https://localhost:5001


NOTE:

https://localhost:5001 URL assumes the default port (5001) that Blazor WebAssembly applications use when running locally. 
If application runs on a different port, you'll need to replace 5001 with the correct port number.
